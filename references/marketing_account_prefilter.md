# 营销号 Prefilter 三级 + 平衡

## L1 (标题级) — **v0.7 放宽: 只跳明显垃圾, 营销号/翻译号留给 conductor AI 过滤**

用户反馈 (2026-05-17): "信噪比不重要, AI 可以剔除. 看不同信噪比下的差异". 所以 mobile L1 **只跳真垃圾**, 多 enter 让 conductor 端 AI 判.

**严跳 (mobile 一定 skip, 浪费 quota)**:
- **代刷 / 账号买卖**: `代刷 / 代肝 / 账号出售 / 转区 / 买码`
- **付费引流**: 标题含 `加 v / 私聊 / 扫码 / 加群` 直接 in title
- **纯网图**: `100 张大图 / 高清美图 / 4K 壁纸合集`
- **跨主题 hashtag 5 连**: `#美股 #A股 #港股 #BTC #黄金` 五种以上 (引流号确认)
- **明显垃圾**: 标题全 emoji / 标题 ≤ 5 字无信息

**放宽 (mobile 不要 skip, 让 conductor AI 判)**:
- ~~数字 listicle (5 步/10 招)~~ → enter, 可能是真用户经验也可能营销号
- ~~翻译号 (Bloomberg/Reuters 中文)~~ → enter, 转载也是行业话术信号
- ~~培训机构~~ → enter, 知道行业 SEO 也是信息
- ~~教程引流~~ → enter, 教程内容真假参半
- ~~硬件测评号 / 网图对比~~ → enter, 数据虽不全但可参考

**结果**: mobile bundle 数从严 prefilter 的 3-4 个涨到 6-8 个, conductor 端 AI 在 Phase C 做最终判定 + 标 A/B/C.

## L2 (内容级, enter 后 OCR 第 1 屏立即判定)

**命中即 back + 移 `_misfires/`** (不计入 5 条上限):
- 段落开头有 `考察点 |` / `解题框架` / `参考答案` 字串 → 题库营销号
- 通用建议 listicle: "第一, 接受自己 第二, 制定计划"
- 私域引流: "扫码 / 加 v / 加微信 / 私聊我"
- 整篇 ChatGPT 排版 (列表 + 加粗 + emoji 锚点) + 通用语气
- 内容只字未提具体使用场景 (针对工具评测主题)

## L3 (作者级)

**降权信号**:
- 头像 + 昵称带 emoji (`情感导师 💜` / `上岸教练 🚀` / `AI 干货 🤖`)
- 主页同模板大量发文
- 收藏数 ≥ 点赞数 (真情流露通常点赞 >> 收藏)

## ⚖️ 平衡 (Test 3 retro 学到, 防过严)

PLAN.md 必须写数量边界:
- **L1 skip 上限**: ≤ 60% search 列表候选
- **L1 enter 下限**: ≥ 5 个候选 (从 search 列表 10+ 标题里)
- **每 keyword 至少 1 bundle**: 5 个 enter 候选全 skip = 过严, relax L1 阈值
- **mobile 不能跨 kw 抛弃**: kw1 一个 bundle 都没 capture → 写 BLOCKER, 不能直接换 kw2

## 真实 OP / A 级信号 (保留 / 优先)

- 标题第一人称 (`我 25 岁` / `我大三` / `那一年我`)
- 标题含具体年龄 / 学位 / 公司名
- 文末有具体时间地点
- 点赞 >> 收藏 (真情流露)
- 评论区有 OP 真实回复 (不是模板)
- 错别字 / 标点不规整 / 真情流露
